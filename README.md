# 🍽️ Zomato Sales Analysis Dashboard

## 📌 Project Overview

This project focuses on analysing Zomato restaurant data to extract meaningful insights about sales performance, customer behaviour, and restaurant trends. The analysis is performed using **SQL, Power BI, and DAX** to build an interactive dashboard that helps in data-driven decision-making.

## 🎯 Objectives

* Analyze overall **revenue performance**
* Track **booking trends and customer behaviour**
* Measure **key KPIs** like ADR, Occupancy Rate, RevPAR
* Identify **top-performing cities and restaurants**
* Understand **cancellation patterns**

---

## 🛠️ Tools & Technologies Used

* **SQL (MySQL)** → Data cleaning & transformation
* **Power BI** → Data visualization & dashboard creation
* **DAX** → KPI calculations
* **Excel/CSV** → Data source


---

## 📈 Dashboard Features

* Revenue trend over time (daily/weekly/monthly)
* City-wise performance comparison
* Top restaurants by revenue
* Booking vs cancellation analysis
* KPI cards for quick insights

---

## 🔍 Business Questions Answered

* Which city generates the highest revenue?
* What is the trend of bookings over time?
* Which restaurants perform the best?
* What percentage of bookings are cancelled?
* How does revenue vary weekly?

---

## 🚀 How to Use

1. Download the dataset (CSV files)
2. Import data into **MySQL** for cleaning
3. Load cleaned data into **Power BI**
4. Create relationships between tables
5. Build DAX measures
6. Design dashboard visuals
